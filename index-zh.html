    About Us

    我們是來自 國立中央大學電機工程學系 National Central University (NCU)的四位學生(已畢業)。 Mspice 是我們的畢業專題作品，由 周求致副教授 指導我們研究與開發。

    在學習電路模擬的過程中，我們發現 Hspice、Ngspice 等工具雖然成熟強大，但對初學者來說常常「功能太多、門檻太高」，還沒開始模擬就先被設定與操作勸退。 於是我們決定用 Python 從零實作一個 SPICE 工具 —— 輕量、好理解、開源，讓更多人能更直覺地學習電路模擬並快速上手。

    Get-started

    Mspice 使用 SPICE netlists—也就是描述電路的純文字檔當作輸入。 需要你列出元件、連接方式，以及加上模擬指令。

    1. 檔案結構

    一個Netlist是一個純文字檔。
    • 第一行: 必須是標題（例如，"* My Circuit"）。它會被Mspice忽略，但有助於你辨識檔案。
    • 中間部分: 列出你的元件 (電阻、電源...)。
    • 最後一行: 必須是 .end 來標記檔案的結束。

    * My Circuit Title R1 1 0 1k V1 1 0 5 .end

    2. 所支援的元件

    其中 Mspice 支援以下元件:
    基本語法: Name Node1 Node2 Value

    • 電阻 (R): R1 1 2 1k (1k Ohm)
    • 電容 (C): C1 2 0 1u (1u Farad)
    • 電感 (L): L1 2 3 10m (10m Henry)
    • 直流電源 DC (V): V1 1 0 dc 5 (5V DC Source)
    • 二極體 Diode (D): D1 1 2 model_name
    • MOSFET (M): M1 D G S B model w=10u l=1u

    *Node 0 永遠是接地 (GND).

    * My Circuit Parts
    R1 1 2 10k
    C1 2 0 1u
    L1 2 3 1m
    * DC Source (5V)
    V1 1 0 dc 5
    * Pulse (Clock Signal)
    V2 3 0 pulse 0 5 1n 1n 1n 10u 20u
    .end

    3. 暫態分析 (Transient Analysis)

    想要看看電壓如何隨著時間變化嗎？使用暫態分析！
    • .tran [step] [stop]
    • Example: .tran 10u 5m 掃描 5ms 的時間，以 10us 的步長
    • 可使用 PULSE 訊號源來產生方波。

    V1 1 0 PULSE(0 5 1m 1n 1n 1m 2m)
    .tran 10u 5m

    4. AC & DC 掃描

    測試電路對不同輸入的反應。 AC Analysis: 檢查頻率響應 (如濾波器)。
    .ac dec 10 1 100k (1Hz 到 100kHz)

    DC Sweep: 檢查特定電壓的響應。
    .dc V1 0 5 0.1 (V1 從 0V 掃到 5V)

    V1 1 0 AC 1
    .ac dec 10 1 100k
    V2 1 0 DC 0
    .dc V2 0 5 0.1
