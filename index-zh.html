    Get-started

    Mspice 使用 SPICE netlists —— 一個描述 元件、節點連線 與 模擬指令 的純文字檔。 需要你列出元件、連接方式，以及加上模擬指令。

    1. Netlist 規則（檔案結構）

    一個 Netlist 包含 4 個部分：
    • 標題: 第一行必須有標題（例如，"* My Circuit"）。它會被Mspice忽略，但有助於你辨識檔案。
    • 元件列表: 一行一個元件。
    • 模擬指令: 三選一 ac/dc/tran。
    • 結尾: 最後一行必須是 .end 來標記檔案的結束。

    * My Circuit Title R1 1 0 1k V1 1 0 5 .end

    2. 元件與電源（支援語法）

    其中 Mspice 支援以下元件:
    基本語法: Name Node1 Node2 Value

    • • 電阻 (R): R<name> <n1> <n2> <value> , 例如: R1 1 2 1k (1k Ohm)
    • • 電容 (C): C<name> <n1> <n2> <value> , 例如: C1 2 0 1u (1u Farad)
    • • 電感 (L): L<name> <n1> <n2> <value> , 例如: L1 2 3 10m (10m Henry)
    • • 二極體 Diode (D): D<name> <n1> <n2> <modelname> , 例如: D1 1 2 model_name
    • • MOSFET (M): M<name> <D> <G> <S> <B> <modelname> w=<width> l=<length> , 例如: M1 D G S B model w=10u l=1u

    *Node 0 永遠是接地 (GND).

    電壓源（三種）：

    • • DC: V<name> <n+> <n-> dc <value>
    • • Pulse: V<name> <n+> <n-> pulse <VL> <VH> <delay> <rise> <fall> <width> <period>
    • • Sin: V<name> <n+> <n-> sin <offset> <amp> <freq> <delay>
    * My Circuit Parts
    R1 1 2 10k
    C1 2 0 1u
    L1 2 3 1m
    * DC Source (5V)
    V1 1 0 dc 5
    * Pulse (Clock Signal)
    V2 3 0 pulse 0 5 1n 1n 1n 10u 20u
    .end

    4. 選擇模擬模式

    Mspice 支援 AC、DC sweep 與 暫態 (Transient)。 三選一，放在 .end 前面即可。

    AC sweep

    ac <variation> <points> <f_start> <f_stop>
    例如：ac dec 10 1 100k

    DC sweep

    dc <source> <start> <stop> <step>
    例如：dc V1 0 5 0.1

    Transient

    tran <step> <stop>
    例如：tran 10u 5m

    * AC 範例 V1 in 0 sin 0 1 1k 0 ac dec 10 1 100k .end

    5. 子電路 Subcircuit（進階）

    想要重複使用一段電路，就用 .subckt 定義，再用 X... 呼叫。 外部節點會依位置做對應；內部節點會自動加上呼叫名稱避免撞名（報告內有 node substitution rule）。

    * 定義 subckt .subckt myRLC a b c R1 a b 1k L1 b int 1m C1 int c 1u .ends * 使用 subckt XU1 n1 n2 0 myRLC .end
